Welcome to Hoogle+

Hoogle+ (pronounced Hoogle Plus) is proof-of-concept tool inspiring to introduce new functionality in Haskell API search engines, most notably Hoogle.

Powered by program synthesis, our contributions (should) include:

  • Search results showing how to compose and combine functions for complex type-transformations
  • Search-by-example as an alternate way of navigating Haskell's API and that of its libraries

The name is a pun on Google+, for the uninitiated. Check us out!

We're still in alpha stage! Expect the unexpected :)
Made with blood, sweat, and tears by Zheng, David, and Nadia.
